# 슬기로운 용돈 생활 — 실행 방법

멀티플레이어 경제 교육 메타버스 게임입니다.
여러 아이가 각자 기기로 접속해 같은 가상 마을에서 동시에 활동합니다.

---

## 1. 준비물

- **Node.js** 설치 (https://nodejs.org → LTS 버전 다운로드)
  설치 후 터미널에서 `node --version` 쳤을 때 숫자가 나오면 OK.

## 2. 처음 한 번만 — 설치

터미널을 열고, 이 폴더로 이동한 뒤:

```
cd 이폴더경로
npm install
```

(예: 바탕화면에 economy-game 폴더가 있으면
`cd Desktop/economy-game`)

## 3. 서버 켜기

```
npm start
```

아래처럼 나오면 성공:

```
슬기로운 용돈 생활 서버 실행 중!
내 컴퓨터에서 접속: http://localhost:3000
```

## 4. 접속하기

- **멘토(선생님) 컴퓨터**: 브라우저에서 `http://localhost:3000`
- **아이들 기기 (같은 와이파이)**:
  1. 서버 켠 컴퓨터의 IP 주소를 확인
     - Windows: 터미널에 `ipconfig` → "IPv4 주소" (예: 192.168.0.5)
     - Mac: `ipconfig getifaddr en0`
  2. 아이들은 브라우저에 `http://192.168.0.5:3000` 처럼 입력
     (192.168.0.5 자리에 위에서 확인한 IP를 넣으세요)

> ⚠️ 반드시 **모두 같은 와이파이**에 연결돼 있어야 합니다.

## 5. 게임 진행 (멘토용)

1. 모든 아이가 이름 입력 후 입장
2. **라운드 1**: 저축 없이 예산 안에서 '꼭 필요한 것' 구매 미션
3. 멘토가 화면 아래 **⚙️ 진행자 메뉴 → "라운드 2 시작"** 클릭
   → 은행이 열림. 저축/이자, 이벤트 칸 활용
4. 마무리 시 **진행자 메뉴 → "게임 종료 & 순위 발표"**
   → 잔액 + 저축 + 이자 합계로 우승자 결정

---

## 설정 바꾸기 (선택)

`server.js` 파일 위쪽 `CONFIG` 부분에서 숫자만 바꾸면 됩니다:
- `START_MONEY`: 시작 예산
- `INTEREST_RATE`: 저축 금리 (0.20 = 20%)

상점 물건·가격·이벤트 내용은 `server.js`의 `ZONES`, `EVENTS` 목록에서 수정.
